# Calypso for Mac — Releases

Public download artifacts for [Calypso for Mac](https://calypso.zncr.pro/download/mac).

This repository holds **only built artifacts**, published automatically by the
release CI in the (private) app repo:

- `Calypso.dmg` — the latest notarized build (attached to each Release)
- `appcast.xml` — the Sparkle auto-update feed
- `latest.json` — version metadata read by the landing page

Don't commit here by hand — everything is generated by the release workflow.
